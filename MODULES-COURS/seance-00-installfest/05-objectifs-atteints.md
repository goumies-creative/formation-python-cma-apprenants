### Terminal et Système de Fichiers
- [ ] **Navigation** : Je peux me déplacer dans l'arborescence avec `cd`
- [ ] **Exploration** : Je sais lister le contenu avec `ls`
- [ ] **Création** : Je peux créer dossiers (`mkdir`) et fichiers (`touch`)
- [ ] **Manipulation** : Je maîtrise copie (`cp`), déplacement (`mv`), suppression (`rm`)
- [ ] **Visualisation** : Je sais afficher le contenu des fichiers (`cat`, `head`, `tail`)

- [ ] **Configuration** : Mon identité Git est configurée (`user.name`, `user.email`)
- [ ] **Initialisation** : Je peux créer un repository avec `git init`
- [ ] **Suivi** : Je vérifie l'état avec `git status`
- [ ] **Sauvegarde** : J'ajoute des fichiers (`git add`) et fais des commits (`git commit`)
- [ ] **Historique** : Je consulte l'historique avec `git log`
- [ ] **Messages** : Mes messages de commit sont clairs et descriptifs

### VS Code - Environnement de Développement
- [ ] **Interface** : Je me repère dans l'interface de VS Code
- [ ] **Navigation** : Je peux ouvrir/éditer/créer des fichiers
- [ ] **Exécution** : Je lance mes programmes Python depuis VS Code
- [ ] **Fonctionnalités** : J'utilise l'auto-complétion et le formatage automatique
- [ ] **Terminal intégré** : Je me sers du terminal dans VS Code

### Python - Premiers Pas
- [ ] **Création** : Je crée des fichiers Python (.py)
- [ ] **Syntaxe base** : Je maîtrise `print()`, `input()`, variables, commentaires
- [ ] **Exécution** : Je lance mes scripts avec `python fichier.py`

## Routine de Travail Recommandée

### Début de session :
```bash
# 1. Ouvrir le terminal
# 2. Se placer dans le projet
cd ~/formation-python-cma

# 3. Ouvrir VS Code
code .

# 4. Vérifier le statut Git
git status
```

### Pendant le travail :
1. Éditer le code dans VS Code
2. Tester régulièrement (python mon_script.py)
3. Vérifier les modifications (git status)
4. Sauvegarder (git add . && git commit -m "...")

### Fin de session :
1. Dernier test du code
2. Dernier commit des modifications
3. Vérification que tout est sauvegardé
4. Planification de la prochaine session
