## Concepts Clés

### Variables et Types
- **Assignation** : `variable = valeur`
- **Types** : `int`, `float`, `str`, `bool`, `list`, `dict`
- **Conversion** : `int()`, `str()`, `float()`

### Manipulation de Texte
- **Méthodes** : `.upper()`, `.lower()`, `.strip()`, `.replace()`
- **Recherche** : `.find()`, `.index()`, `in`
- **Découpage** : `.split()`, `.join()`

### Slicing et Formatage
- **Indexation** : `texte[0]`, `texte[-1]`
- **Slices** : `texte[2:5]`, `texte[:3]`, `texte[3:]`
- **Formatage** : f-strings, `.format()`
