## Concepts Clés

### Structures Conditionnelles
- **if/elif/else** : prise de décision
- **Conditions simples** : comparaisons basiques
- **Conditions composées** : and, or, not

### Opérateurs
- **Comparaison** : `==`, `!=`, `<`, `>`, `<=`, `>=`
- **Logiques** : `and`, `or`, `not`
- **Appartenance** : `in`, `not in`

### Formatage
- **f-strings** : `f"texte {variable}"`
- **.format()** : `"texte {}".format(variable)`
- **Spécificateurs** : alignement, padding, précision
