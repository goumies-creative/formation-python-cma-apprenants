# Guide GitHub Codespaces - Mairie de Paris

## Accès à votre environnement de cours

### 1. Créer un compte GitHub
- Allez sur [github.com](https://github.com)
- Cliquez sur "Sign up"
- Utilisez votre email professionnel

### 2. Accepter l'invitation
- La formatrice vous enverra une invitation par email
- Cliquez sur "Accept invitation"

### 3. Lancer votre environnement

![Lancement Codespaces](https://docs.github.com/assets/cb-11427/images/help/codespaces/new-codespace-button.png)

1. Allez sur le repository du cours
2. Cliquez sur le bouton vert "Code"
3. Sélectionnez l'onglet "Codespaces"
4. Cliquez sur "Create codespace on main"

### 4. Premier lancement
- Attendez 2-3 minutes pendant l'installation
- Vous verrez un environnement VS Code complet dans le navigateur
- Python et tous les outils sont pré-installés

## Interface de Codespaces

- **Terminal complet** en bas (PowerShell/Bash)
- **Explorateur de fichiers** à gauche
- **Éditeur de code** au centre
- **Extensions VS Code** pré-installées

## Sauvegarde de votre travail

### Méthode 1 : Commit Git
```bash
git add .
git commit -m "Mon travail session X"
git push
```

### Méthode 2 : Téléchargement
- Clic droit sur un fichier → "Download"
- Ou glisser-déposer depuis l'explorateur

##  Gestion des Codespaces
### Arrêter proprement
- Menu "Codespaces" en bas à gauche → "Stop Current Codespace"

### Reprendre le travail
- [github.com/codespaces](https://github.com/codespaces)
- Cliquez sur votre codespace

### Économiser les heures
- Pensez à arrêter votre codespace après chaque session
- Les codespaces s'arrêtent automatiquement après 30 minutes d'inactivité

## Dépannage
### Le terminal ne répond pas
- Menu Terminal → New Terminal

### Codespace lent
- Vérifiez votre connexion internet
- Redémarrez le codespace

### Problème d'installation
- Contactez la formatrice avec une capture d'écran
