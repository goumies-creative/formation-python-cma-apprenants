# Guide GitPod - Alternative Rapide

## Accès instantané à votre environnement

### 1. Prérequis
- Avoir un compte GitHub (voir guide Codespaces)
- Visiter [gitpod.io](https://gitpod.io) une fois pour lier les comptes

### 2. Lancement express
- **Méthode 1** : Allez sur `https://gitpod.io/#/votre-repo-url`
- **Méthode 2** : Installez l'extension "GitPod" dans votre navigateur

### 3. Premier lancement
- Environ 30 secondes de démarrage
- Environnement complet pré-configuré
- Tous les outils Python installés

## Avantages de GitPod

- **Plus rapide** que Codespaces
- **Terminal complet** sans limitations
- **150 heures gratuites** par mois
- **Pré-builds** pour démarrage instantané

## Gestion des travaux

### Sauvegarde automatique
- GitPod commit automatiquement vos changements
- Pour sauvegarder : poussez vers GitHub

### Téléchargement de fichiers
- Glisser-déposer depuis votre ordinateur
- Ou utiliser l'explorateur de fichiers

## Commandes utiles

```bash
# Vérifier l'environnement
python TEST-ENVIRONNEMENT.py

# Lancer un serveur Flask
python -m flask run

# Ouvrir un notebook Jupyter
jupyter notebook
```

## Support
- Problème de compte : contactez la formatrice
- Environnement lent : vérifiez votre connexion
- Erreur d'installation : redémarrez le workspace
